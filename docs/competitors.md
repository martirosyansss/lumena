# Конкурентный анализ MedBridge Tourism

Медицинский туризм РФ/СНГ → Армения (Ереван). Документ — результат многоагентного
исследования: 51 кандидат найден, 16 разобраны по живым источникам + отдельный профиль Bookimed.

> Достоверность: цифры конкурентов взяты с их сайтов/Trustpilot и могут расходиться/устаревать.
> Пункты с пометкой «(требует проверки)» нуждаются в отдельной сверке. Дата сбора — июнь 2026.

---

## 1. Краткое резюме

- **Поле прямых RU→Армения фасилитаторов перегрето, но все слабы там, где MedBridge Tourism сильна.**
  MedTravel, MedWay, Med-ArmTours — та же модель и направление, но ни один не даёт одновременно:
  JCI/ISO/Минздрав РА + флагманские клиники поимённо + смету «под ключ» + явную бесплатность +
  узкую заточку под РФ. Это и есть свободная позиция MedBridge Tourism.
- **Главная структурная угроза — не фасилитаторы, а сами клиники-партнёры** (Astghik, Nairi,
  Erebouni): владеют аккредитациями напрямую, имеют опубликованные цены и тысячи отзывов,
  зовут пациентов РФ напрямую — в обход посредника, которому MedBridge Tourism платит комиссию.
- **В ЭКО/суррогатстве MedBridge Tourism конкурирует с глубокими нишевиками** (Embrycare, Эмбримама,
  Arni, Stork) — у них убойные цифры результата и собственные базы сурмам/доноров.
- **Прозрачность цены — системная дыра рынка и главный недоиспользованный рычаг.**
  Большинство армянских игроков цены не публикуют; калькулятор есть только у альтернативных
  направлений и части репро-агентств.
- **Bookimed перехватывает поисковый трафик, но по Армении мелок** (только пластика Еревана).
- **Главный относительный дефицит MedBridge Tourism — молодой бренд без публичных цифр доверия.**

## 3. Профиль Bookimed (детально)

- **Позиционирование:** «Платформа №1 по подбору лечения за рубежом» — глобальный
  маркетплейс-агрегатор с врачом-координатором.
- **Масштаб:** с 2014 (Киев; офисы Лондон/Стамбул/Гонконг); ~1 500–1 700 клиник, 50 стран,
  ~750 000–1 000 000+ пациентов/запросов. Цифры между страницами расходятся — маркетинговая
  агрегация, а не выверенные данные.
- **Модель:** бесплатно для пациента, заработок на комиссии от клиник (как у MedBridge Tourism).
  Калькулятора нет (требует проверки); показывают вилки цен, финальную — через заявку.
- **Армения:** второстепенное направление, почти только пластика Еревана. Стоматология,
  чек-апы, ЭКО/суррогатство по Армении системно не покрыты — прямая ниша MedBridge Tourism.
- **Доверие:** Trustpilot «Excellent» 4.6/5 (~915–1024 отзыва), верификация отзывов по
  транзакции; сертификаты GHA/TEMOS/ISO; рейтинги клиник/врачей.
- **Угроза для MedBridge Tourism: MEDIUM** (high в верхней части поисковой воронки, low в ядре MedBridge Tourism).
- **Перенять:** верифицированные отзывы, SEO-страницы «[процедура] в Армении: цена», мультиканал.
- **Отстроиться:** фиксированная смета «под ключ» ДО поездки; глубина по Армении; безвиз+близость;
  координатор на земле; честность цифр.

## 4. Бенчмарк MedBridge Tourism против рынка

**Сильнее:** институциональное доверие (единственный фасилитатор на связке JCI/ISO/Минздрав РА +
клиники поимённо) · заточка под боль РФ (безвиз ЕАЭС + близость + −30–60% к Москве) ·
нейтральный «проводник» против клиник, продающих себя · широкий портфель + легальное ЭКО для РФ.

**Отстаёт:** нет публичных цифр трек-рекорда · до недавнего — нет прозрачной цены до контакта ·
нет внешних рейтингов (Trustpilot/Google) и живых видеоотзывов · слабее нишевиков в репродукции ·
узкий RU-трафик против 5–6-язычных.

## 5. Белые пятна и возможности

1. Прозрачный калькулятор сметы «под ключ» для Армении (никто из прямых фасилитаторов не имеет).
2. «JCI/ISO принадлежат нашим клиникам — отбираем только аккредитованные».
3. Легальное суррогатство/ЭКО для граждан РФ через Армению — отдельная сильная посадочная.
4. Конкретные цифры экономии по процедурам «Москва частная vs Ереван».
5. «Нейтральный проводник на стороне пациента» + гарантия урегулирования конфликтов.
6. Цифры доверия + видеоотзывы пациентов из РФ.
7. Логистический козырь «безвиз + 2,5–3 ч» как визуальный блок-сравнение vs Турция/Корея.
8. Мессенджер-first захват + лид-магнит-гайд.

---

## 6. Угрозы

1. Обход посредника партнёрами (Astghik/Nairi/Erebouni зовут РФ напрямую) — №1.
2. Демпинг прозрачностью цены (Astghik/Arni/UAmedTOURS показывают цифру сразу).
3. Перехват репро-лидов нишевиками (Embrycare с гео-SEO по РФ).
4. SEO-доминирование Bookimed в верхней воронке.
5. Якорь «первопроходца» MedTravel.
6. Узкий RU-фокус = меньше входящего трафика против мультиязычных.
